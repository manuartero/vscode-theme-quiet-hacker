# Quiet Hacker

Hard black. Pure green. Nothing else.

A minimal VS Code theme for vibe coding. No distractions, no rainbow syntax, just you and the machine.

## Palette

| Role | Hex |
|---|---|
| Background | `#000000` |
| Green | `#00FF41` |
| Dark Green | `#00802a` |
| Text | `#E0E0E0` |
| Muted | `#666666` |

## Philosophy

Most themes use 15+ colors for syntax highlighting. Quiet Hacker uses 4. Keywords and constants glow green. Strings go dark green. Comments fade to grey. Everything else is white.

Borders are pure green. The terminal is green. Your cursor is green. You're in the matrix now.

## Install

Search **"Quiet Hacker"** in the VS Code Extensions Marketplace, or:

```
ext install manuartero.quiet-hacker
```
